# keiba_blood_analysis

競馬の血統分析を行うプロジェクトです。

## 概要

このリポジトリは、競馬における血統データの分析を目的としています。

## 機能

- 血統データの収集
- 血統パターンの分析
- レース成績との相関分析

## セットアップ

```bash
git clone https://github.com/moeki43/keiba_blood_analysis.git
cd keiba_blood_analysis
```
